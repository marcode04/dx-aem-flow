# Documentation

Full documentation is available on the [KAI website](https://easingthemes.github.io/dx-aem-flow/).

To run locally: `cd website && npm install && npm run dev`

This directory contains machine-readable reference files used by plugin skills at runtime:

| File | Purpose |
|------|---------|
| `reference/skill-catalog.md` | All skills with invocation syntax and outputs |
| `reference/agent-catalog.md` | All agents with tools, models, and descriptions |
| `reference/config-reference.md` | `.ai/config.yaml` full schema |
| `TODO.md` | Internal planning and future improvements |
