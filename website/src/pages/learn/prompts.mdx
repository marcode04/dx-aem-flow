<SectionHeading
  badge="Rules"
  badgeColor="warning"
  title="Rules (.claude/rules/)"
  subtitle="Path-scoped convention files that auto-load when you edit matching files."
  bg="alt"
/>
<Section>
  <CommandBlock label="Example: .claude/rules/fe-javascript.md">{`---
paths: ["**/*.js"]
applyTo: "**/*.js"
---
# JavaScript Conventions
- Use CustomComponent custom elements (extends HTMLElement)
- Lifecycle: options() -> beforeLoad() -> afterLoad()
- No ES modules -- use Gulp concat pipeline
- Externalize selectors to component config`}</CommandBlock>

  <HighlightBox severity="info" title="Dual Frontmatter">
    Rules need both <code>paths:</code> (Claude Code) and <code>applyTo:</code> (Copilot CLI)
    to work in both tools. Single file, dual format.
  </HighlightBox>
</Section>
