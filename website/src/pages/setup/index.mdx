{/* Step 1 */}
<SectionHeading
  badge="Step 1"
  badgeColor="secondary"
  title="Install Plugins"
  subtitle="Add the shared marketplace and install the plugins your project needs."
  bg="primary"
/>
<Section>
  <CommandBlock label="Claude Code CLI">{`# Add marketplace (one-time)
/plugin marketplace add easingthemes/dx-aem-flow

# Install plugins
/plugin install dx-core@dx-aem-flow
/plugin install dx-aem@dx-aem-flow          # AEM projects only
/plugin install dx-hub@dx-aem-flow          # Multi-repo orchestration only`}</CommandBlock>

  <CommandBlock label="GitHub Copilot CLI">{`# Same commands — same marketplace, same plugins
/plugin marketplace add easingthemes/dx-aem-flow
/plugin install dx-core@dx-aem-flow
/plugin install dx-aem@dx-aem-flow          # AEM projects only`}</CommandBlock>

  <HighlightBox severity="info" title="Updating">
    <code>/plugin marketplace update dx-aem-flow</code> updates all installed plugins to the latest version.
  </HighlightBox>
</Section>

  <div class="grid md:grid-cols-3 gap-4 mt-6">
    <ContentCard icon="mdi:numeric-1-circle" iconColor="bg-emerald-600" title="Minimal">
      Skip this step entirely. <code>/aem-init</code> auto-generates a basic <code>component-index.md</code>. Agents work but guess at architecture.
    </ContentCard>
    <ContentCard icon="mdi:numeric-2-circle" iconColor="bg-amber-500" title="Recommended">
      Create <code>project.yaml</code> + <code>file-patterns.yaml</code> + <code>architecture.md</code>.
      Agents find files reliably and understand your project.
    </ContentCard>
    <ContentCard icon="mdi:numeric-3-circle" iconColor="bg-brand-700" title="Full">
      All files above + <code>content-paths.yaml</code> + <code>features.md</code> + <code>component-index-project.md</code>. Full multi-repo, multi-brand capabilities.

  </div>
